### 调试类

log（debug用）

```
log.Println(...)
```



报错

```go
if err != nil {
	panic("failed to connect database, err = " + err.Error())
}
```

一般错误直接写在 if 里：

```go
if err := myfunc(); err != nil {
    ...
}
```

如果含有其他返回参数，且在之后的作用域中用到，则写在 if 外面：

```go
res, err := http.Get(url)
if err != nil {
    ...
}
```



创建一个error对象

```
err := errors.New("hello,error")
```



自定义error

```go
//一个包裹了错误类型对象的自定义错误类型
type MyError struct {
	err error 
}

//订制Error()
func (e MyError) Error() string {
    return e.err.Error()
}
```
